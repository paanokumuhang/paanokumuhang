PhilHealth ID

Ang PhilHealth ID at PhilHealth Identification Number (PIN) ay mahalagang requirement para sa mga serbisyong pangkalusugan sa Pilipinas. Ang PIN ay isang lifetime number na ginagamit sa lahat ng PhilHealth transactions tulad ng hospital admission, benefit claims, at membership verification.

Step-by-Step Guide

  • Mag-fill out ng PhilHealth Member Registration Form (PMRF)
    I-download ang PMRF mula sa opisyal na website ng PhilHealth o kumuha ng kopya sa pinakamalapit na PhilHealth branch. Punan ang lahat ng kinakailangang detalye tulad ng buong pangalan, address, civil status, at dependents (kung mayroon). Siguraduhing tama at malinaw ang impormasyon.
  • Pumunta sa pinakamalapit na PhilHealth office
    Dalhin ang napunuan na PMRF sa PhilHealth branch at magtungo sa Member Services section para sa submission at initial checking.
  • Magpresenta ng valid ID at supporting documents
    Ipakita ang kahit isang valid government-issued ID. Para sa first-time registrants, maaaring hingin ang PSA-issued Birth Certificate bilang karagdagang patunay ng pagkakakilanlan.
  • Hintayin ang verification at issuance ng PhilHealth Number (PIN)
    Iva-verify ng PhilHealth staff ang iyong mga dokumento. Kapag kumpleto at walang isyu, agad na ibibigay ang iyong PhilHealth Identification Number (PIN).
  • Humiling ng PhilHealth ID Card (optional)
    Matapos makuha ang iyong PIN, maaari kang humiling ng PhilHealth ID card kung available sa branch:
